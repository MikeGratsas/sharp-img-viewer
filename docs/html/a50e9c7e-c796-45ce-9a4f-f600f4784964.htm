System Requirements

This topic describes hardware and software requirements to use the Sharp IMG Viewer 2008.

Hardware Requirements

Recommended hardware

  • Intel® Pentium® processor, or an AMD Opteron, AMD Athlon64 or AMD Athlon XP processor 400 MHz (800 MHz+ recommended)

  • 128 MB RAM (256 MB recommended)

  • 50 MB hard disk space

  • 800 x 600 256 colors display (1024 x 768 High Color - 16-bit recommended)

  • mouse or pointing device

Software Requirements

Sharp IMG Viewer 2008 is developed for the .NET Framework 2.0 or later.

Supported operating systems
