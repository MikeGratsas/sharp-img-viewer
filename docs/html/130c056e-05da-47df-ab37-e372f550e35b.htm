Sharp IMG Viewer Help

This topic describes how to uninstall the Sharp IMG Viewer 2008.

Prior to the Sharp IMG Viewer 2008 removal you should quit all launched instances of the application.

Uninstall

There are two methods to uninstall the Sharp IMG Viewer 2008:

  1. Go to Start->Control Panel->Add or Remove Programs, find Sharp IMG Viewer 2008 and then press Remove button.

  2. Go to Windows menu: Start->Programs->Sharp IMG Viewer 2008 and click Uninstall Sharp IMG Viewer 2008 shortcut.
