Open Image

Sharp IMG Viewer Help

The dialog allows you to open image files for use in the Sharp IMG Viewer 2008.

Image files can be opened once the Sharp IMG Viewer 2008 is started by choosing Open... from the File menu, by clicking toolbar button, or by dragging image files onto the main window.

If you check Preview checkbox, a single click on an image filename will display a small version.
