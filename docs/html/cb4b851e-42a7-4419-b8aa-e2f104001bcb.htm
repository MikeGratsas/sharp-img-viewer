The following resources related to the Sharp IMG Viewer 2008 are available:

Links to Resources

  • This software uses the FreeImage open source graphics library. More details can be found from this location.

  • Zip, GZip, Tar and BZip2 compression library #ZipLib originally developed by Mike Krueger for use with a free IDE for .NET #Develop.

  • PdfSharp .NET library for processing Adobe PDF files under MIT License.

    Copyright © 2005-2019 empira Software GmbH, Cologne (Germany). All rights reserved.

  • DiscUtils .NET library to read and write ISO files and Virtual Machine disk files (VHD, VDI, XVA, VMDK, etc.) developed by Kenneth Bell under MIT License.

  • Json.NET is a popular high-performance JSON framework for .NET developed by James Newton-King.

  • ZXing.Net is a library which supports decoding and generating of barcodes ported from java by Michael Jahn.

  • log4net library helpful to enable logging at runtime which is part of the Apache Logging Services.

  • You probably already have it, but if not, the .NET Framework 2.0 can be downloaded from Microsoft Download Center.

  • Although not needed by the Sharp IMG Viewer 2008, you can obtain the .NET Framework 3.5 Service Pack 1 from Microsoft Download Center.

  • The Microsoft Windows Image Acquisition (WIA) Automation Layer 2.0 is a full-featured image manipulation component that provides end-to-end image processing capabilities. You can locate this component from Microsoft Documentation.

  • This software uses the FFmpeg open source library to import video files. More details can be found from this location.

  • To read information about the features of 7z and LZMA compressed file formats designed by Igor Pavlov please see 7-Zip.
