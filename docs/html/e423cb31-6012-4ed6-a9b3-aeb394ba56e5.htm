Sharp IMG Viewer Help

The topic describes how to purchase the Sharp IMG Viewer 2008 license.

Your trial version will expire in 15 days after you first install the product.

If you do not renew (activate) the trial version that is about to expire, you will not be able to use the Sharp IMG Viewer 2008. It is necessary to purchase the license to make the program work.

Purchasing Information

If you wish to buy a perpetual license, all you need to do is make a payment to the author.

You can order the Sharp IMG Viewer 2008 at BMT Micro, Inc. website.

For more information please check out the Sharp IMG Viewer 2008 website.
