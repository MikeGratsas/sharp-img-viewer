Import Images

Sharp IMG Viewer Help

This topic describes how to load images from various sources

Images can be imported from:

  • the assembly by choosing File->Import->From Assembly... menu item;

  • the Zip, GZip, BZip2, LZip, Tar, Z, PPMd, 7z, LZMA, XZ, RAR, ARJ, XAR, CPIO, Zstandard, Brotli, Snappy, LZ4, LZ5, Lizard, LZH, LZO, LZFSE, UNIX dump, Microsoft Cabinet, Encrypted APPX and WIM compressed files as well as ISO, FAT, NTFS, SquashFS, Btrfs, XFS disk image files by choosing File->Import->From Archive... menu item;

  • the VMware VMDK, Oracle VirtualBox VDI, Microsoft VHD and VHDX, Apple DMG, XenServer Virtual Appliance XVA virtual machine disk files and Simple Deployment Image files by choosing File->Import->From Virtual Disk File... menu item;

  • the Windows resources in PE, NE, LE format files by choosing File->Import->From Executable... menu item;

  • the Shockwave Flash files by choosing File->Import->From Flash Movie... menu item;

  • the video files by choosing File->Import->From Video File... menu item;

  • the compound and compiled help files by choosing File->Import->From Compound File... menu item;

  • the RPM package files by choosing File->Import->From Package File... menu item;

  • the resource files by choosing File->Import->From Resource... menu item.
